What is Poke Labs?

Poke Labs is a collection of self-hosted, zero-dependency services for AI agents. Every service is a single Python file using only the standard library. No npm. No pip install. No Docker required (but supported).

Services

🔗 Link Preview

Extract title, description, and Open Graph images from any URL. Free tier: 3/day. Unlimited via x402 USDC payments.

POST /api/preview

🤖 Poke Bot

GitHub webhook handler. Auto-labels issues by priority (P0-P3) and PRs by size (S/M/L/XL). Responds to !poke commands.

POST /webhook

💰 Billing Engine

Subscriptions, usage tracking, and invoicing. Free/Pro/Team plans. x402 payments on Base.

POST /api/subscribe

📦 Package Registry

Publish, search, and version packages. Think npm but minimal. JSON-backed.

POST /api/publish

🧩 Skills Hub

Discover and install skills for your agent. All MIT licensed.

GET /api/skills

📊 Pricing API

Real-time quotes for Poke Labs services. Transparent pricing.

GET /api/quote

Quick Start

# Clone and run any service — zero dependencies
git clone https://github.com/pokelabshq/poke.git
cd poke
python3 poke status
python3 poke start link-preview
