Services

All services are free to use with a daily limit. Built with Python, MIT licensed.

→ Barcode Generator

Generate Code39, Code128, and EAN-13 barcodes as SVG or JSON.

GET /api/generate

→ URL Shortener

Shorten long URLs. Custom aliases supported.

POST /api/shorten

→ Sentiment Analysis

Analyze text sentiment with confidence scores.

POST /api/analyze

→ Link Preview

Extract title, description, and image from any URL.

POST /api/preview

→ QR Generator

Generate QR codes from any text or URL.

GET /api/generate

→ Hash Generator

Generate MD5, SHA-1, SHA-256, SHA-512 hashes.

GET /api/hash

→ UUID Generator

Generate UUID v4, v1, and v7 identifiers.

GET /api/uuid

→ Timestamp Converter

Convert Unix, ISO 8601, and human-readable dates.

GET /api/convert

→ JSON Formatter

Format, validate, and minify JSON.

POST /api/format

→ Base64 Tool

Encode and decode Base64 text and binary.

POST /api/encode

→ Markdown Renderer

Render Markdown to HTML with GFM support.

POST /api/render

→ Password Generator

Generate secure random passwords.

GET /api/generate
